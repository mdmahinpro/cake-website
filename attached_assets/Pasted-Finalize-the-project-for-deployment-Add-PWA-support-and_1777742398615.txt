PWA SETUP:
Install: npm install vite-plugin-pwa

Update vite.config.js:
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import { VitePWA } from 'vite-plugin-pwa'

export default defineConfig({
  plugins: [
    react(),
    VitePWA({
      registerType: 'autoUpdate',
      manifest: {
        name: 'Sweet Dreams Cakes',
        short_name: 'CakeShop',
        description: 'Custom handcrafted cakes for every occasion',
        theme_color: '#d4a574',
        background_color: '#1a0a00',
        display: 'standalone',
        start_url: '/',
        icons: [
          { src: '/icon-192.png', sizes: '192x192', type: 'image/png' },
          { src: '/icon-512.png', sizes: '512x512', type: 'image/png' },
        ],
      },
      workbox: {
        globPatterns: ['**/*.{js,css,html,ico,png,svg}'],
      },
    }),
  ],
  base: '/',
  build: { outDir: 'dist', sourcemap: false },
})

Create README.md:
# Sweet Dreams Cake Website

## Quick Deploy

### Vercel (Free, Recommended)
1. Push to GitHub
2. Connect repo to vercel.com
3. Add environment variable: VITE_ADMIN_PASSWORD=yourpassword
4. Deploy - done!

### Netlify (Free)
1. npm run build
2. Drag dist folder to netlify.com/drop
3. Add env variable in site settings

### Hostinger Shared Hosting
1. npm run build on your computer
2. Upload ALL contents of dist folder to public_html
3. Upload .htaccess to public_html
4. Done! Visit your domain.

## Admin Panel
- URL: yourdomain.com/control
- Password: set VITE_ADMIN_PASSWORD environment variable

## First Time Setup
1. Visit /control, login
2. Go to Settings - update shop name, WhatsApp number, Facebook page
3. Delete demo gallery items
4. Add your real cake photos
5. Toggle WhatsApp/Facebook as preferred order channel
